Git Notes

Git Commands Details

Clone and Status

  • For cloning a repository in our machine use =>

    "git clone "URL"

  • Status command shows status of our files =>

    "git status"


4 types of Status of Files in git

  1. Untracked Files

    => Newly Created but git does not know about it.

  2. Modified Files

    => Changes are updated into git repository

  3. Unmodified

    => Unchanged Files

  4. Staged

    => File is ready to commited

Add & Commit

  • Add new or changed Files in your Working Directory to staging area

    git add (file_Name)

  • Commit is used to keep record of changing file data

    git command -m "Type a message"
